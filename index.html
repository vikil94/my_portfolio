Projects

Buddflix

Buddflix is a movie recomendation website that gives you a movie based off the type of cannabis you are using, using two APIs to gather this information. The front-end was built in react which was the portion of the project that I worked on

LuxxGame

An interactive, technology-enhanced version of the childhood classic “tag,” LUXX participants play the game in a large, dark room wearing custom-designed, LED-powered suits. Once the game begins, each suit begins to pulse with one of two colours, dividing the players into two teams. Participants must attempt to tag players on the other team. The game ends when all players are the same colour, or when the time has run out.

Perspective

Perspective is a 2D puzzle platform game where the player must change their view to get past each level. This game was created in Unity, my main focus on the project was to create the game logic as well as the animation state changes

DemonBrawler

Built in Unity, DemonBrawler is a two person 2D competitive combat game. Players fight swarms of demons, players can choose to fight the demons on their own level or superpunch the demon to their partners. The game uses PS4 controllers to play. The main focus of this project was to implement the game logic properly while the focus on animation came after we had properly figured out the logic
